API Trade In - Explication générale

Cette API gère l'ensemble du tunnel d'estimation, de cotation (Argus) et de diagnostic dynamique propre à chaque appareil pour Comparecycle.

Structure globale du flux
  • Catalog/argus : Récupération de l'ID Argus de référence.
  • Criteria/questions : Questions dynamiques conditionnelles basées sur l'appareil.
  • Simulation/calculate : Calcul instantané de la valeur de rachat théorique.

API Trade In - Credentials

Entête HTTP requis : X-AUTH-CR

Client ID (TradeIn Preprod)

GET /Catalog/argus

Récupère l'identifiant ou la valeur de référence argus en cours de validité.

GET
https://api-preprod.comparecycle.com/v1/Catalog/argus

Query Parameters

ParamètreTypeDescription
idargusint (optional)Filtre sur un identifiant précis.
argus_by_timestampstring (optional)Filtre l'argus à une date passée précise.

GET /Criteria/questions

Récupère le questionnaire de diagnostic dynamique et conditionnel lié à l'appareil sélectionné.

GET
https://api-preprod.comparecycle.com/v1/Criteria/questions

API Partenaire - Explication générale

Cette API vous permet de configurer, lister et lier vos boutiques physiques ou vos comptes d'apport d'affaires tiers.

API Partenaire - Credentials

Partner Secret Token

GET /Stores/list

Retourne la liste complète de vos magasins physiques enregistrés pour la collecte en point de vente.

GET
https://api-preprod.comparecycle.com/v1/Stores/list

API Logistique - Explication générale

Permet d'automatiser l'envoi d'appareils rachetés en générant des étiquettes de transport (Mondial Relay, Chronopost, Colissimo) et en suivant les statuts logistiques.

API Logistique - Credentials

Logistics Public Key

POST /Shipments/label

Génère un fichier de transport au format PDF ou ZPL pour l'expédition de l'appareil par le client final.

POST
https://api-preprod.comparecycle.com/v1/Shipments/label

API Pandas - Explication générale

Fournit des agrégats de données massives (Big Data) pour l'édition de bilans RSE et l'analyse de l'empreinte carbone évitée grâce aux reprises d'appareils.

API Pandas - Credentials

Pandas Analytics Key

GET /Metrics/sustainability

Récupère le volume total de CO2 économisé, le poids de DEEE évité sur une période donnée.

GET
https://api-preprod.comparecycle.com/v1/Metrics/sustainability

API B2B - Explication générale

Destinée aux entreprises souhaitant faire estimer et revendre une flotte complète de parcs informatiques ou de téléphones d'entreprise (Rachats en masse).

API B2B - Credentials

B2B Enterprise Token

POST /Bulk/quote

Soumet une liste d'appareils (via un tableau JSON de structures) pour obtenir une cotation globale B2B.

POST
https://api-preprod.comparecycle.com/v1/Bulk/quote
