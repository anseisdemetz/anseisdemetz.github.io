Explication générale

Bienvenue sur l'API de Trade-in (reprise) de Comparecycle. Cette API en marque blanche vous permet d'intégrer notre tunnel d'estimation et de rachat directement sur vos plateformes ou applications partenaires.

Fonctionnement du flux

Le parcours standard se déroule selon les étapes suivantes :

  1. Authentification : Récupération de votre token via le endpoint /auth/login.
  2. Catalogue : Exploration des marques (Brand) et des catégories (Category).
  3. Évaluation : Récupération des questions de diagnostic dynamiques propres à l'appareil.
